# Layer Mode Contract

`layer_mode_contract_v1` freezes the canonical Layer Mode vocabulary.

## Canonical Modes

- `shadow`
- `assist`
- `enforce`

## Meaning

- `shadow`: compute and log only
- `assist`: softly influence effective output
- `enforce`: allow direct policy-strength enforcement

## Principle

Layer Mode values describe influence strength only.
They do not control whether semantic computation exists.
