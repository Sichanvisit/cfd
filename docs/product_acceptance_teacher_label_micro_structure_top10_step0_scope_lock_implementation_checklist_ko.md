# Teacher-Label Micro-Structure Top10 Step 0 범위 고정 체크리스트

## 목적

이 문서는 `Step 0. 계산 범위 고정`이 실제로 끝났는지 확인하기 위한 체크리스트다.

## Checklist

### A. Top10 범위 확정

- [ ] Top10 항목 10개가 확정되었다
- [ ] 항목 이름이 문서 간 동일하다
- [ ] `state25 마스터 플랜`과 `Top10 상세 기준서`와 `로드맵`이 같은 목록을 사용한다

### B. 구현 방식 분류 확정

- [ ] `그대로 사용 + 점진 보강` 항목이 확정되었다
- [ ] `보강 후 승격` 항목이 확정되었다
- [ ] `anchor 추가 후 재구축` 항목이 확정되었다

### C. 각 항목의 재료 확정

- [ ] `body_size_pct_20` 주 재료가 문서에 명시되었다
- [ ] `upper_wick_ratio_20` 주 재료가 문서에 명시되었다
- [ ] `lower_wick_ratio_20` 주 재료가 문서에 명시되었다
- [ ] `doji_ratio_20` 주 재료가 문서에 명시되었다
- [ ] `direction_run_stats` 주 재료가 문서에 명시되었다
- [ ] `range_compression_ratio_20` 주 재료가 문서에 명시되었다
- [ ] `volume_burst_decay_20` 주 재료가 문서에 명시되었다
- [ ] `swing_high_retest_count_20` 주 재료가 문서에 명시되었다
- [ ] `swing_low_retest_count_20` 주 재료가 문서에 명시되었다
- [ ] `gap_fill_progress` 주 재료와 anchor가 문서에 명시되었다

### D. 제외 범위 확정

- [ ] 1차에서 하지 않을 항목이 문서에 적혔다
- [ ] 신규 대형 이벤트 엔진을 이번 scope에서 제외했다
- [ ] tick-by-tick 전체 저장을 이번 scope에서 제외했다

### E. 다음 단계 연결

- [ ] Step 1 owner가 명시되었다
- [ ] Step 2 owner가 명시되었다
- [ ] Step 3 이후 export 경로가 명시되었다
- [ ] 1차 구현 5개 우선순위가 고정되었다
