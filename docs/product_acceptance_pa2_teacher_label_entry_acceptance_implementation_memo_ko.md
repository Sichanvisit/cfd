# Product Acceptance PA2 Teacher Label Entry Acceptance Implementation Memo

## 상태

이 문서는 teacher-label 기준으로 PA2를 다시 여는 kickoff memo다.

기존 baseline에서는 `must_enter = 0`이라 PA2가 실질 종료처럼 보였지만,
이번 스크린샷 3장은 `실제로는 들어갔어야 하는 자리`가 더 있었다는 점을 보여준다.

즉 이 memo의 목적은
`queue 0 = 충분`
이 아니라
`teacher label 기준 entry miss를 줄인다`
로 PA2 목표를 다시 정의하는 것이다.
