## 상태

이 문서는 teacher-label 기준으로 PA4를 다시 정렬하는 kickoff memo다.

현재 PA4는 이미 여러 sub-axis가 구현돼 있지만,
스크린샷 3장은 그 구현을 단순히 수치가 아니라
`행동 정답` 기준으로 다시 평가해야 한다는 점을 보여준다.

즉 이 memo의 목적은

- current PA4 patch chain을 유지하면서
- 남은 backlog를 teacher-label release/cut 기준으로 다시 좁히는 것

이다.
