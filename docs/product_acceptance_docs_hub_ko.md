# Product Acceptance Docs Hub

## 목적

이 문서는 `product_acceptance*.md` 계열 문서가 너무 많아졌을 때,
어디서부터 읽고 어떤 문서를 기준 문서로 봐야 하는지 한눈에 정리하기 위한 허브다.

- 기존 작은 문서는 그대로 둔다
- 링크는 깨지지 않게 유지한다
- 대신 `무엇을 먼저 읽을지`와 `어떤 문서는 memo만 보면 되는지`를 여기서 정리한다
